## Customer Churn Prediction using Machine Learning

## Project Overview
Customer churn is one of the biggest challenges for subscription-based businesses. This project predict whether a customer will leave the company using Machine Learning classification algorithm.

## Objective 
Develop a predictive model that helps businesses identify customers likely to churn so they can improve custoner retention.

## Dataset 
The dataset contain customer demographic information, account details, subscription information, and churn labels.
Typical features:
- Gender
